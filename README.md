# YuGiOh-discord-bot
유희왕 마스터 듀얼 헬퍼 봇입니다.
>>> 주요 기능
> 우라라 및 유령토끼 타이밍 조회 (구현중)
> 유희왕 카드 조회 기능 (미구현)

## 개발환경 설정
``` bash
$ pip3 install python3 // 파이썬 설치 

$ pip3 install discord // Linux or Mac
$ pip install -U discord // Window

$ pip install -r requirements.txt // 개발환경 복사
$ python3 main.py // 실행
```
